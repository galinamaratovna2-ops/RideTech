# RideTech
Аналитика сервиса заказа такси: анализ операций, спроса, влияния различных факторов на ключевые метрики сервиса. Формулировка рекомендаций, которые учитывают интересы и пассажиров, и водителей.

1. RideTech_1 - Анализ поездок аудитории сервиса и выявление скрытых паттернов в них.

Ключевые показатели: количество, среднее расстояние, среднее время, выручка
Анализ отмен по количеству и причинам
Рассмотрение характеристик поездок, выявление выбросов, подозрительных паттернов.

2. RideTech_2 - Построение гео-heatmap спроса для сервиса такси
Код
  - агрегирует спрос по районам и часам по количеству поездок, средней стоимости, среднему расстоянию.
  - строит heatmap по количеству поездок
  - анализирует топ-3 пиковых часа для каждого района
